# CDC COVID-19 Dashboard

A modular Python toolkit for ingesting, processing, analyzing, visualizing, and summarizing CDC COVID-19 mortality data.

## Features

- **Data Ingestion:** Download most recent CDC data as CSV.
- **Preprocessing:** Clean, filter, and structure data for analytics.
- **Analysis:** Aggregate COVID deaths by year, state, age group, and more.
- **Visualization:** Trend charts and plots using matplotlib.
- **AI Summarization:** Narrative dashboard summaries with HuggingFace Transformers.
- **Extensible Structure:** Add new features or modules by editing separate Python files.
